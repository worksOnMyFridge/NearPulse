ШАГ 1: Отправьте Procfile в Git
════════════════════════════════

Дважды кликните файл: deploy-railway.bat

(Батник закоммитит и запушит Procfile автоматически)

═══════════════════════════════════════════════════════

ШАГ 2: Настройте Railway Start Command
═══════════════════════════════════════

1. Откройте: https://railway.app
2. Проект NearPulse → Settings
3. Deploy → Custom Start Command
4. Введите: node src/api.js
5. Save

═══════════════════════════════════════════════════════

ШАГ 3: Redeploy
═══════════════

1. Railway → NearPulse → Deployments
2. Кликните три точки → Redeploy

═══════════════════════════════════════════════════════

⏱️ Подождите 1-2 минуты

═══════════════════════════════════════════════════════
