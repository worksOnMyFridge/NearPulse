Q: Безопасно ли запускать cleanup.bat?
A: ✅ ДА! Скрипт только перемещает файлы в /archive
   Ничего не удаляется безвозвратно

Q: Что если мне понадобится старый документ?
A: 📦 Все старые файлы будут в /archive
   Вы сможете их найти там

Q: Нужно ли делать backup перед очисткой?
A: Git - это ваш backup! Но если хотите:
   git add . && git commit -m "Before cleanup"

Q: Сломается ли что-то после очистки?
A: ❌ НЕТ! Удаляются только:
   - Временные файлы (backup, test)
   - Старые документы → в /archive
   - Устаревшие скрипты → в /archive
   
   Весь актуальный код остается!

Q: Можно ли отменить очистку?
A: ✅ ДА! Используйте git:
   git checkout .
   (если не закоммитили)

Q: Что делать если cleanup.bat выдает ошибку?
A: Выполните команды вручную из SECURITY_AUDIT.md
   раздел "ДЕЙСТВИЯ"
