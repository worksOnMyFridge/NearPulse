### Fail-Safe Architecture | –ù–µ—É–±–∏–≤–∞–µ–º–∞—è –ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–∞

NearPulse is built to survive. Every external API call has:
- **Multi-source fallback** ‚Äî if CoinGecko is down, we try Intear, then Ref Finance, then Nearblocks
- **Smart caching** ‚Äî data is cached for 30 seconds to reduce load and speed up response
- **Graceful degradation** ‚Äî if an API fails, the app still works with available data
- **Rate limit awareness** ‚Äî automatic throttling to respect API limits

### As a Developer | –î–ª—è –†–∞–∑—Ä–∞–±–æ—Ç—á–∏–∫–æ–≤

```bash
# Clone and install
git clone https://github.com/worksOnMyFridge/NearPulse.git
cd NearPulse

# Bot setup
```
